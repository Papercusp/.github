# Contributing

Thanks for your interest! A couple of things worth knowing before you dive in:

## These are extracted mirrors, not the primary source

Repositories in the Papercusp organization are generally published as standalone,
domain-free ("borrowable") libraries extracted from an internal monorepo. Day-to-day
development happens internally and is synced out to these repositories. Because of
that:

- **Issues** are the best way to report a bug, ask a question, or propose a change —
  please open one, we do read them.
- **Pull requests** are welcome, but a PR opened directly against one of these
  repositories may need to be ported into the internal source of truth before it can
  land, rather than being merged here directly. We'll follow up on your PR to explain
  how it will get incorporated — please bear with the extra step.

## Reporting a security issue

Please see [SECURITY.md](./SECURITY.md) instead of a public issue or PR.

## Code of conduct

Be respectful and constructive. We want this to be a welcoming place for anyone who
finds these libraries useful.
